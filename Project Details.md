# Object Detection for Blind Assistant

## 1. Project Title

**Object Detection for Blind Assistant Using Artificial Intelligence and Computer Vision**

---

## 2. Problem Statement

Visually impaired individuals face difficulties in identifying objects, obstacles, and important surroundings while moving independently. Traditional assistance methods such as walking sticks and guide dogs provide limited information about the environment. There is a need for an intelligent system that can detect surrounding objects in real time and provide audio feedback, enabling visually impaired users to navigate safely and confidently.

---

## 3. Project Overview

The Object Detection for Blind Assistant is an AI-powered assistive system designed to help visually impaired people understand their surroundings. The system captures real-time images through a camera, detects objects using a deep learning model, and converts the detected object names into speech output.

The application improves independent mobility by informing users about nearby objects such as people, vehicles, chairs, doors, bottles, and other obstacles. The solution combines Computer Vision, Deep Learning, and Text-to-Speech technologies to provide a practical and user-friendly assistance tool.

---

## 4. Project Objectives

1. To develop a real-time object detection system for visually impaired users.
2. To identify and classify surrounding objects accurately.
3. To provide voice-based feedback using Text-to-Speech technology.
4. To improve user safety and independent navigation.
5. To create a low-cost and portable assistive solution.
6. To enhance accessibility through Artificial Intelligence.

---

## 5. System Modules

| Module No | Module Name           | Description                                                     |
| --------- | --------------------- | --------------------------------------------------------------- |
| Module 1  | Image Acquisition     | Captures real-time images through the device camera.            |
| Module 2  | Object Detection      | Detects and identifies objects using a trained AI model (YOLO). |
| Module 3  | Object Classification | Classifies detected objects into predefined categories.         |
| Module 4  | Audio Feedback        | Converts object labels into speech output.                      |
| Module 5  | User Interface        | Displays system status and controls.                            |
| Module 6  | Alert System          | Provides immediate alerts when obstacles are detected.          |
